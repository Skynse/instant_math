Evaluate the following definite integral:

∫0π/2 sin5(x) cos3(x) dx

Step-by-Step Solution

Guided derivation using u-substitution

1

Identity Application

Rewrite the cosine term using the Pythagorean identity cos²(x) = 1 - sin²(x) to prepare for substitution.

= ∫ sin⁵(x) (1 - sin²(x)) cos(x) dx
2

Substitution Variable

Let u = sin(x), then du = cos(x) dx. Adjust the limits of integration.

  • When x = 0, u = 0
  • When x = π/2, u = 1
3

Integration

Integrate the polynomial form with respect to u.

∫₀¹ (u⁵ - u⁷) du = [ u⁶/6 - u⁸/8 ]₀¹
