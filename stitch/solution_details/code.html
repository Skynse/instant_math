Current Problem

Solve for x:

3(x - 5) + 4 = 2x + 8
Final Result
x = 19
check_circle Verified Logic

Logical Breakdown

1

Expand the expression

Apply the distributive property to the left side: 3 * x and 3 * (-5).

3x - 15 + 4 = 2x + 8
2

Combine like terms

On the left side, simplify -15 + 4 to get -11.

3x - 11 = 2x + 8
3

Isolate variable terms

Subtract 2x from both sides to move all x-terms to the left.

(3x - 2x) - 11 = 8
x - 11 = 8
4

Final Isolation

Add 11 to both sides to find the final value of x.

x = 8 + 11
x = 19
